# stk-addon-retriever

This simple utility allows you to get all Addons from the SuperTuxKart Addons API.

## Requirements
You will need the following dependencies:
* curl
* pv (for progress)
* pick (install with `python -m pip install pick`)
* supertuxkart (duh) 

## How2Use???

Clone this repo then execute:
```
python __main__.py
```
